# Matrix Operations in C

## About
This is a simple C program that performs basic matrix operations:
* Addition
* Multiplication
* Transpose

It uses functions and 2D arrays for better understanding.

---

## How to Run
1. Compile the program:
```
gcc matrix.c -o matrix
```
2. Run the program:
```
.\matrix
```
