# Matrix Operations in C

## About
This is a simple C program that performs basic matrix operations:
* Addition
* Multiplication
* Transpose

It uses functions and 2D arrays for better understanding.

---

## How to Run
1. Compile the program:
```
gcc matrix.c -o matrix
```
2. Run the program:
```
matrix
```
---

## Example Input
```
2 2
2 2

1 2
3 4

5 6
7 8
```

---

## Example Output

```
Addition:
6 8
10 12

Multiplication:
19 22
43 50

Transpose of A:
1 3
2 4
```
